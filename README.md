# 🛒 E-Commerce Sales Analysis (SQL Project)

## 📌 Project Overview
This project analyzes e-commerce sales data using SQL to uncover insights about revenue performance, customer behavior, and sales trends.

---

## 🎯 Objectives
- Analyze revenue across product categories
- Identify top customers and calculate Customer Lifetime Value (CLV)
- Detect and remove duplicate records
- Understand geographic sales distribution
- Explore monthly revenue trends

---

## 🧰 Tools & Skills Used
- SQL (Data Cleaning, Aggregation, Joins)
- Excel (Dataset storage)
- Data Analysis
- Business Insights

---

## 📂 Dataset
The dataset contains transactional e-commerce data including:
- Orders
- Customers
- Products
- Revenue
- Locations
